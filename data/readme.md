# Data Directory

This folder contains the sample accelerometer data used in the project demonstration.

## Contents

- **PullUps_Dips.csv**  
  Raw accelerometer measurements for pull-ups and dips.

- **PushUps_Curls.csv**  
  Raw accelerometer measurements for push-ups and biceps curls.


## Data Format

All data files follow this general structure:

- `timestamp` 
- `acc_x` — Acceleration on X-axis
- `acc_y` — Acceleration on Y-axis
- `acc_z` — Acceleration on Z-axis


## Purpose

It is used exclusively for:
- demonstrating the pipeline,
- testing preprocessing steps,
- generating plots and metrics.
